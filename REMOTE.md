# Remote 101

### Forward SSH Key

```
ssh -A pivotal@WORKSTATION.OFFICE.pivotal.io
```

### Lastpass (lpass)

To disable the pop-up box:

```
$ type lpass-ssh
lpass-ssh is aliased to `LPASS_DISABLE_PINENTRY=1 lpass'
$ lpass-ssh USERNAME@pivotal.io
```
